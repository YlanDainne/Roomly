# Roomly (RentBuddy) 🏡

**Roomly** (also known as RentBuddy) is a beautifully designed web platform crafted to help students find available, affordable, and secure renting places and university homes. With an emphasis on "Aura" aesthetics, quiet luxury, and elegant minimalism, it provides a premium, noise-free experience for students seeking secure long-term contracts.

## ✨ Features

- **Architectural UI Interface:** Ditching generic tech layouts in favor of an editorial, high-end "Aura" design, featuring *Playfair Display* & *Inter* typography.
- **Trust & Verification System:** Highlights Verified Landlords, Secure Contracts, Student Filters, and Campus Proximity checking.
- **Cinematic Experience:** Atmospheric layout featuring smooth, high-end CSS animations, dark-themed overlays, and cinematic lighting.
- **Responsive Navigation:** Lightning-fast page transitions using React Router DOM.

## 🛠️ Built With

- **[React.js](https://reactjs.org/)** (Bootstrapped with `create-react-app`)
- **[React Router](https://reactrouter.com/)** - For seamless client-side routing.
- **[Lucide React](https://lucide.dev/)** - For elegant, crisp vector icons.
- **Pure CSS3** - Custom animations, overlays, and responsive "glass/card" design systems.

## 🚀 Getting Started

Follow these simple instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YlanDainne/Roomly.git
   cd Roomly/roomly
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   *Your browser will automatically open to [http://localhost:3000](http://localhost:3000) showcasing the application!*

## 🎨 Design Philosophy

Roomly abandons typical chaotic real-estate interfaces. It utilizes deep cinematic tinting, stark architectural cards, and fine hairline borders—mirroring the serene, high-end experience of finding the perfect university home in locations like Cebu.
